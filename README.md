**Описание:** Практическая работа по обнаружению сетевых хостов, сканированию портов и анализу уязвимостей операционных систем Windows и Linux.

**Ключевые задачи:**
- Настройка сетевого взаимодействия между виртуальными машинами
- Обнаружение хостов в сети с использованием ARP-сканирования
- Полное сканирование портов и определение версий сервисов
- Анализ уязвимости FTP-сервера vsftpd 2.3.4

- Сравнение подходов к безопасности различных ОС

**Технологии:** Kali Linux, Metasploitable3, Nmap, Metasploit

**Результаты:**
- Обнаружено и проанализировано 25+ сетевых сервисов
- Выявлена и протестирована критическая уязвимость в FTP-сервере
- Составлен перечень уязвимых конфигураций для Windows и Linux систем
- Документирован полный цикл тестирования безопасности
